SOLAR SYSTEM SIMULATION

This simulation shows the orbital periods between each planet. This is an accurate simulation of orbital periods for each planet relative to each other. A year is defined as the length of time a planet takes to complete one full rotation around the sun. Here, it is easy to see that as distance from the sun increases, so does the orbital period. The orbital period for each planet is as follows:

  • Mercury: 87.97 days (0.2 years)
  • Venus : 224.70 days (0.6 years)
  • Earth: 365.26 days(1 year)
  • Mars: 686.98 days(1.9 years)
  • Jupiter: 4,332.82 days (11.9 years)
  • Saturn: 10,755.70 days (29.5 years)
  • Uranus: 30,687.15 days (84 years)
  • Neptune: 60,190.03 days (164.8 years)
